Tools¶

Helper Functions

This module contains a variety of functions for manipulating images in ways do NOT return a modified version of the original image.

align_image_with_openpnm(im)

Rotates an image to agree with the coordinates used in OpenPNM.

bbox_to_slices(bbox)

Given a tuple containing bounding box coordinates, return a tuple of slice objects.

extend_slice(s, shape[, pad])

Adjust slice indices to include additional voxles around the slice.

extract_subsection(im, shape)

Extracts the middle section of a image

extract_regions(regions, labels[, trim])

Combine given regions into a single boolean mask

extract_cylinder(im[, r, axis])

Returns a cylindrical section of the image of specified radius.

extract_subsection(im, shape)

Extracts the middle section of a image

fftmorphology(im, strel[, mode])

Perform morphological operations on binary images using fft approach for improved performance

find_outer_region(im[, r])

Finds regions of the image that are outside of the solid matrix.

get_border(shape[, thickness, mode, …])

Creates an array of specified size with corners, edges or faces labelled as True.

get_planes(im[, squeeze])

Extracts three planar images from the volumetric image, one for each principle axis.

insert_cylinder(im, xyz0, xyz1, r)

Inserts a cylinder of given radius onto a given image

insert_sphere(im, c, r[, v, overwrite])

Inserts a sphere of a specified radius into a given image

in_hull(points, hull)

Test if a list of coordinates are inside a given convex hull

make_contiguous(im[, keep_zeros])

Take an image with arbitrary greyscale values and adjust them to ensure all values fall in a contiguous range starting at 0.

mesh_region(region[, strel])

Creates a tri-mesh of the provided region using the marching cubes algorithm

norm_to_uniform(im[, scale])

Take an image with normally distributed greyscale values and convert it to a uniform (i.e.

overlay(im1, im2, c)

Overlays im2 onto im1, given voxel coords of center of im2 in im1.

ps_ball(r[, smooth])

Creates spherical ball structuring element for morphological operations

ps_disk(r[, smooth])

Creates circular disk structuring element for morphological operations

ps_rect(w, ndim)

Creates rectilinear structuring element with the given size and dimensionality

ps_round(r, ndim[, smooth])

Creates round structuring element with the given radius and dimensionality

pad_faces(im, faces)

Pads the input image at specified faces.

randomize_colors(im[, keep_vals])

Takes a greyscale image and randomly shuffles the greyscale values, so that all voxels labeled X will be labelled Y, and all voxels labeled Y will be labeled Z, where X, Y, Z and so on are randomly selected from the values in the input image.

seq_to_satn(seq)

Converts an image of invasion sequence values to saturation values.

size_to_seq(size[, bins])

Converts an image of invasion size values into sequence values.

subdivide(im[, divs, overlap, flatten])

Returns slices into an image describing the specified number of sub-arrays.

zero_corners(im, pad_width)

Fills corners of a padded-image with 0, given a pad width and list of faces such as [“left”, “front”] at which the image has been padded.
