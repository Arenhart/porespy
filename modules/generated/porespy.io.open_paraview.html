open_paraview¶

open_paraview(filename=None, im=None, **kwargs)[source]¶

Open a paraview state file or image directly in paraview.

Parameters

  • filename (str) – Path to input state file.

  • im (ND-array) – An image to open directly. If no filename given, then this image is sent to to_paraview and a state file is created with a random name. Any additional keyword arguments are sent to to_paraview.
