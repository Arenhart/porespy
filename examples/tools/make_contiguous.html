## Documentation for make_contiguous

*make_contiguous(im, keep_zeros=True)**

ake an image with arbitrary greyscale values and adjust them to ensure ll values fall in a contiguous range starting at 0.

his function will handle negative numbers such that the most negative umber will become 0, unless keep_zeros is True in which case t will become 1, and all 0’s in the original image remain 0.

### Parameters im : array_like An ND array containing greyscale values keep_zeros : Boolean If True (default) then 0 values remain 0, regardless of how the other numbers are adjusted. This is only relevant when the array contains negative numbers, and means that -1 will become +1, while 0 values remain 0.

### Returns image : ND-array An ND-array the same size as im but with all values in contiguous orders.
